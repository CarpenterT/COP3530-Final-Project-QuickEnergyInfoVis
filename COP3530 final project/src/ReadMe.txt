In this src folder:

energy.data: raw data for the energy.py library.
energy.py: library from the CORGIS website, allowing for a more direct extraction of data.
ls.py: file containing some constant information and helper functions for the main python file.
main.cpp: file that contains the main function of the sorting algorithms.
source.h: file that contains the sorting algorithms themselves.
sortData.exe: file called by the main python file to sort data. It is created using the makefile.
